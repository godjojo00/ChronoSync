# ChronoSync - 共用行事曆

ChronoSync 是一個共用行事曆應用，旨在幫助團隊成員協調和計劃活動。

## 功能

- 共用行事曆
- 事件增刪查改
- 團隊成員之間的事件同步
- [其他功能...]

## 開始使用

### 前置要求

- Node.js
- Go
- MongoDB

### 安裝步驟

1.開前端
```
git clone 
cd chronosync
```
2.
```
cd frontend
```
3.
```
npm install
```
4.
```
npm run dev
```
5.開後端
```
cd ..
cd backend
```
6.
```
go mod tidy
```
7.
```
go run main.go
```
確保mongodb已啟動
